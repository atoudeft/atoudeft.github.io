Collaboration

Dans le monde réel, des objets collaborent entre eux pour réaliser des tâches. Ainsi, un client d’une banque peut créditer son compte bancaire avec un montant d’argent. Un enseignant peut évaluer le travail d’un étudiant pour un cours et lui attribuer une note. Une personne peut conduire une voiture et la déplacer d’un point A à un point B.

De manière similaire, dans un programme orienté objet, des objets collaborent pour fournir les fonctionnalités du programme. Par exemple, si une classe Document représente un document électronique (fichier PDF, image JPEG, …) :

public class Document {

//...

}

On peut avoir dans une classe Imprimante, une méthode imprimer() qui permet d’imprimer un document :

public class Imprimante {

//...

public boolean imprimer(Document doc) {

//...

}

}

Dans un programme :

Document monCv = new Document();

Imprimanter laserJet = new Imprimante();

if (laserJet.imprimer(monCv))

System.out.println("CV imprimé avec succès");

On peut avoir des objets Personne qui déplacent des voitures :

public class Voiture {

//...

public void avancer(int distance) {

//...

}

}

public class Personne {

//...

public void deplacer(Voiture v, int distance) {

v.avancer(distance);

}

}

Dans un programme :

Personne annie = new Personne();

Voiture maToyota = new Voiture();

annie.deplacer(maToyota, 30);

Avant de déplacer la voiture, on peut s’assurer que la personne possède la bonne catégorie de permis de conduire :

public class Voiture {

private char categoriePermisNecessaire;

//...

public void getCategoriePermisNecessaire() {

return categoriePermisNecessaire;

}

public void avancer(int distance) {

//...

}

}

public class Personne {

private char categoriePermis;

//...

public boolean deplacer(Voiture v, int distance) {

if (categoriePermis==v.getCategoriePermisNecessaire()) {

v.avancer(distance);

return true;

}

else {

return false;

}

}

}

Dans un programme :

Personne annie = new Personne();

Voiture maToyota = new Voiture();

if (!annie.deplacer(maToyota, 30))

System.out.println("La voiture ne peut pas être déplacée");

Il y a collaboration dès qu’un objet utilise un autre objet. Généralement, la collaboration s’effectue par « envoi de message ». Dans des langages tels que Java, l’envoi d’un message s’effectue par appel de méthode : envoyer un message à un objet consiste à appeler une méthode de l’objet.
