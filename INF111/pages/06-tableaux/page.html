Introduction

Un tableau est une structure de données composée de cases ou cellules ou éléments contenant des données de même type et de même nature.

Le tableau est identifié par un nom qui permet ainsi d’accéder à toutes les données du tableau.

Les cases du tableau possèdent chacune un numéro appelé indice qui permet de les distinguer les unes des autres. En Java, la première case possède toujours l’indice 0 (figure 1).

Figure 1. Tableau tabNotes contenant 10 cases de type double.

Création

1. Déclaration

En Java, la déclaration d’un tableau se fait d’une des 2 façons suivantes :

double[] tabNotes;

ou

double tabNotes[];

La déclaration crée la variable de type tableau mais ne crée pas de tableau.

2. Instanciation

Un tableau est un objet. Sa création se fait donc à l’aide de l’opérateur new :

tabNotes = new double[35];

Dans cet exemple, tabNotes référence un tableau de 35 cases de type double indexées de 0 à 34.

La taille du tableau (nombre de cases) est spécifiée à la création du tableau et ne peut plus changer.

Un tableau possède un attribut public length qui contient la taille du tableau. Ainsi, tabNotes.length désigne la taille du tableau tabNotes.

La taille du tableau peut être une valeur définie à l’exécution. Par exemple :

int nbNotes;

System.out.print("Combien de notes avez-vous ? ");

nbNotes = clavier.nextInt();

tabNotes = new double[nbNotes];

où clavier est l’objet de type Scanner permettant de lire au clavier (tel que vu en classe).

3. Initialisation

Il est possible de créer un tableau en fournissant ses données initiales au lieu de fournir la taille :

double[] tab = {25.6, 12.3, 11.44, 31.25, 62.35};

Cet exemple crée un tableau de 5 cases contenant les valeurs fournies entre les accolades { … }.

L’initialisation ne peut se faire que lors de la déclaration du tableau. Il n’est pas possible de la faire après la déclaration :

double[] tab;

tab = {25.6, 12.3, 11.44, 31.25, 62.35}; //NON

L’ancienne syntaxe d’initialisation (qui peut être utile à l’occasion) est :

double[] tab = new double[]{25.6, 12.3, 11.44, 31.25, 62.35};

Manipulation

1. Accès aux éléments

L’accès à une case du tableau se fait à l’aide de l’indice de la case mis entre crochets. Ainsi, tabNotes[0] est la première case du tableau et tabNotes[5] est la 6e case.

On peut, par exemple, afficher le contenu d’un tableau à l’aide d’une boucle :

for (int i=0; i<tabNotes.length; i++)

System.out.println(tabNotes[i]);

2. Affectation

Puisqu’un tableau est un objet, l’affectation de tableau est une affectation de références, comme pour les autres objets (figure 2) :

double[] tabNotes = new double[10];

double[] tab2;

tab2 = tabNotes;

Figure 2. L’affectation de tableaux est une affectation de références.

Dans cet exemple, tabNotes et tab2 référencent le même tableau. Ainsi, tabNotes[5] et tab2[5] désignent la même donnée.

Tableau d’objet

Un tableau d’objets est un tableau dont chaque case contient une référence vers un objet.

Supposons qu’on a une classe CompteBancaire (comme celle vue en classe) :

public class CompteBancaire {

//…

}

L’instruction suivante déclare une variable de type tableau d’objets CompteBancaire :

CompteBancaire[] tComptes;

L’instruction suivante crée un tableau de 15 objets CompteBancaire (mais, attention, aucun objet CompteBancaire n’est créé) :

tComptes = new CompteBancaire[15];

Pour remplir le tableau, il faut instancier la classe CompteBancaire pour créer les objets CompteBancaire.

Par exemple :

tComptes[0] = new CompteBancaire();

CompteBancaire cpt = new CompteBancaire();

tComptes[1] = cpt;

for (int i=2; i<tabNotes.length; i++)

tabNotes[i] = new CompteBancaire();

Il peut arriver (même si ce n’est souvent pas souhaitable) que 2 cases du tableau référencent le même objet CompteBancaire :

tComptes[7] = tComptes[1];

Les cases 7 et 1 du tableau tComptes référencent le même objet CompteBancaire.

Tableau multidimensionnels

1. Tableaux 2D

Un tableau à 2 dimensions permet d’organiser les données en lignes et colonnes, comme une matrice mathématique (figure 3).

Figure 3. Tableau à 2 dimensions.

En Java, la déclaration se fait avec 2 paires de crochets [ ] :

double[][] matrice;

Et l’instanciation :

matrice = new double[6][4];

qui crée un tableau de 6 lignes et 4 colonnes.

L’accès aux éléments se fait à l’aide de 2 indices, un pour la ligne et un pour la colonne. Ainsi, matrice[3][2] désigne l’élément se trouvant sur la 4e ligne et la 3e colonne du tableau matrice.

En réalité, un tableau à 2 dimensions est un tableau à 1 dimension dont chaque case contient un tableau à 1 dimension.

Il en résulte que les lignes d’un tableau à 2 dimensions ne sont pas obligées d’être toutes de même taille.
